We use font icons in our app. Icon fonts are just fonts. However, instead of containing letters or numbers, they contain symbols and glyphs. You can style them with CSS in the same way you style regular text. They are defined & used as any other font.
Adding a new font icon
Whenever we add a new icon we'll get a 6 font files from design. The font files can be found under blink/app/resources/font/thoughtspot_21.
  1. thoughtspot-iconfont-regular-webfont.eot
  2. thoughtspot-iconfont-regular-webfont.png
  3. thoughtspot-iconfont-regular-webfont.svg
  4. thoughtspot-iconfont-regular-webfont.svg
  5. thoughtspot-iconfont-regular-webfont.woff
  6. thoughtspot-iconfont-regular-webfont.woof2
2 step process
Step 1: Update files and folder - Developer should replace the old files with new files. Make sure to keep the file names same and update the number in folder. Ex: thoughtspot_21 -> thoughtspot 22.
Step 2: Create new icon class - Follow the pattern of any already defined icon class. Give a meaningful name to the class which represents the icon not it usage and update the content. The new icon will have a new unique content character associated with it. You can find that from thoughtspot-iconfont-regular-webfont.png.
