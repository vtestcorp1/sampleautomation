Layout templates
Positioning:
Static
By default all elements are static.
Relative
Class name - pr
Used when we want to position an element relative to its original position. You use the positioning attributes top, left bottom and right to set the location.
Absolute
Class name - pa
Used when we want to position an element relative to body(top left corner of the page). You use the positioning attributes top, left bottom and right to set the location. These values will be relative to the next parent element with relative (or absolute) positioning.
Fixed
Class name - pf
It's positioned relative to the viewport, or the browser window itself. The viewport doesn't change when the window is scrolled.
Tip: In most cases pa is used on a child element when it's parent has pr.
<div class="pr">Relatively positioned element
<div class="pa">
Absolutely positioned element
</div>
</div>
Floats
Float is a CSS positioning property. Page elements with the CSS float property applied to them are just like the images in the print layout where the text flows around them. Floated elements remain a part of the flow of the web page.
Left
Class name - fl
Floats the element to left.
Right
Class name - fr
Floats the element to right.
Clearfix
Class name - clearfix
Applied on the parent element and it uses CSS pseudo selector (:after) to clear floats.
Example:
<div class="clearfix">
<div class="fl">
Float left this element
</div>
<div class="fr">
Float right this element
</div>
</div>
Display:
None
Class name - dn
Totally removes the element from the page.
Block
Class name - db
By default (without setting a width) they take up as much horizontal space as they can and adds a break at the end. Elements like <div>, <ul>, <p>, <section> are block elements by default.
Inline
Class name - di
Elements like <span>, <em>, <b> are block elements by default. Wrapping text in those elements within a string of text doesn't break the flow of the text. An inline element will accept margin and padding but ignore height and width. Margin and padding will only push the elements horizontally away.
Inline block
Class name - dib
Similar to inline except that you are able to set a width and height.
